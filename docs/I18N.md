# I18N

## Implementation

For internationalization, I used `next-intl`.
The app currently includes three languages:

## Locales

- `en`
- `it`
- `de`

## Routing

Locale routing is configured with:

- `i18n/routing.ts`
- `proxy.ts`

Locale is always in URL path (for example `/en`, `/it`, `/de`).

## Messages

Messages for all languages are stored in the i18n messages folder:

- `i18n/messages/en.json`
- `i18n/messages/it.json`
- `i18n/messages/de.json`

## Language Switcher

Users can change the language using the Select component in the header.
