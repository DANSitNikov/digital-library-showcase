# I18N

## Locales

- `en`
- `it`
- `de`

## Routing

Locale routing is configured with:

- `i18n/routing.ts`
- `proxy.ts`

Locale is always in URL path (for example `/en`, `/it`, `/de`).

## Messages

Message files are in:

- `i18n/messages/en.json`
- `i18n/messages/it.json`
- `i18n/messages/de.json`
